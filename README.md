# 马原题库练习系统

《马克思主义基本原理》刷题练习平台，支持按章节练习、错题记录、进度保存和本地数据管理。

## 项目结构

- `docs/`：题库原始来源，按章节保存 Markdown 文档。
- `Web/index.html`：前端入口页面。
- `Web/js/app.js`：页面渲染、答题逻辑、错题本和本地进度管理。
- `Web/js/data/`：前端运行时题库数据，由 `docs/` 中的 Markdown 内容整理生成。

## 题库说明

题库以 `docs/` 目录下的 Markdown 文件为依据，包含单选题、多选题和判断题三类题型。题目、选项、答案和排序均按原始文档整理；多选题答案若分散在多个括号中，会合并为同一道题的答案。

每道题都包含解析字段。选择题解析会结合题干、正确选项和排除项说明作答依据；判断题解析会说明命题成立或错误的判断原因。

## 使用方式

直接双击 `Web/index.html` 即可在浏览器中使用。答题进度保存在浏览器本地存储中，不会上传到服务器。

## 数据维护约束

- `docs/` 是题库原始依据，除非需要修订源文档，否则不要直接修改。
- 前端使用的数据位于 `Web/js/data/`。
- 修改题库数据时，应保持章节顺序和题型顺序与 `docs/` 一致：单选题、多选题、判断题。
- 修改后应检查数据能正常加载，并确认题目 ID、答案和选项格式有效。
